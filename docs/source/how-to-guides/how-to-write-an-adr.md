# Architecture Decision Records (ADRs)

An Architecture Decision Record (ADR) documents a significant architectural choice, the context
that led to it, and the consequences of the decision.

## How to Create a New ADR

1. Copy `adr-NNN-template.md` (found [here](../references/templates/adr-NNN-template.md)) to `adr-NNN-<kebab-case-title>.md`
2. Fill in all sections
3. Add a row to the index table and the toctree [index.md](../explanations/adrs/index.md)
4. Open a PR
